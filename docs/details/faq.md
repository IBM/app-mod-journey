# Frequently Asked Questions - Application Modernization

## TECHNOLOGY AND TOOLS

### I have CloudPak4Apps, am I entitled to use AMA?

NO, CloudPak for Apps is not entitled to use AMA. AMA is a standalone tool .


### What entitlement do I need to use AMA?

Following are the product name where you are entitled to use AMA:
IBM Enterprise Application Runtimes
IBM Modernized Runtime Extension for Java
